# Cortex

A modern, premium landing page for **Cortex** — an AI productivity startup.  
*Your second brain, powered by AI.*

## Live preview

Open `index.html` in any browser (double-click or drag into Chrome, Edge, Firefox). No build step or server required.

## Tech stack

- **HTML5** — Semantic markup
- **Tailwind CSS** (CDN) — Styling
- **Vanilla JavaScript** — Navbar scroll, smooth scroll, fade-in on scroll (Intersection Observer)

No frameworks, no build tools. Single-file layout.

## Sections

1. **Sticky navbar** — Transparent → white with shadow on scroll  
2. **Hero** — Headline, CTA buttons, product preview card  
3. **Social proof** — Trusted by 10,000+  
4. **Features** — Capture instantly, AI organization, Find anything fast  
5. **Product demo** — Fake dashboard UI  
6. **How it works** — 3 steps  
7. **CTA** — Get started  
8. **Footer** — Links and copyright  

## Deploy on GitHub Pages

1. Create a new repository on GitHub.
2. Upload `index.html` and `README.md`.
3. Go to **Settings → Pages**.
4. Under **Source**, choose **Deploy from a branch**.
5. Select branch `main` and folder **/ (root)**.
6. Save. Your site will be at `https://<username>.github.io/<repo-name>/`.
